# 常见问题解答（FAQ）

本文汇总了用户使用 Ceru Music（澜音）过程中最常遇到的问题及解决方法，便于快速自助排查。

---

## 一、安装与启动

**Q1：Windows 安装时提示「此应用可能危害你的设备」，是病毒吗？**

A：不是。由于 Ceru Music 目前尚未购买 Windows 代码签名证书，系统对未签名程序会显示警告。本软件完全开源，发布包由 GitHub Actions 自动化打包，可通过官方 GitHub 仓库验证。安装时点击右侧三个点 → 保留 → 点击下拉按钮 → 仍然保留，即可正常安装。

**Q2：macOS 提示「澜音已损坏，无法打开」，如何处理？**

A：这是 macOS 的 Gatekeeper 签名校验机制导致的，非真正损坏。请打开终端，执行以下命令后重新启动应用：

```bash
sudo xattr -r -d com.apple.quarantine /Applications/澜音.app
```

执行时需要输入 macOS 用户密码。

**Q3：应用启动后一片空白或界面无响应？**

A：尝试以下方法：① 完全退出后重启应用；② 检查是否有多个实例在运行；③ 删除应用缓存目录后重新启动（具体路径请参考官方 Q 群中的说明）。

## 二、插件与音源

**Q4：打开应用后搜索不到任何歌曲？**

A：Ceru Music 本身不内置音乐曲库，需要先安装音源插件才能使用在线搜索功能。请进入「设置 → 插件 / 音源」，添加并启用兼容的插件（如澜音插件或洛雪插件）。

**Q5：插件已安装，但搜索结果仍为空？**

A：① 确认插件已在「设置 → 插件」中点击「使用」激活；② 当前选择的音源与插件版本不兼容，尝试切换到其他音源；③ 检查网络是否正常，部分音源需要良好的网络连接。

**Q6：插件更新后功能异常，如何处理？**

A：尝试在「设置 → 插件」中禁用后重新启用该插件；若问题持续，可尝试删除该插件并重新安装最新版本。

---

## 三、播放问题

**Q7：点击播放后没有声音，音量图标显示正常？**

A：① 确认系统音量未静音；② 进入「设置 → 播放 → 音频输出」，确认输出设备选择正确：③ 如果使用外接音频设备，先确认设备已正确连接并被系统识别；④ 关闭并重新打开应用。

**Q8：播放时频繁卡顿或缓冲？**

A：① 检查网络连接速度；② 尝试降低播放音质（如从无损切换到标准品质）；③ 清理应用缓存；④ 关闭全屏播放中的背景动画和音频可视化，减少 CPU 占用。

**Q9：歌词不显示或显示乱码？**

A：该歌曲可能由音源插件提供的歌词数据为空或编码不正确，属于数据问题，无法在应用内修正。可尝试切换到其他音源搜索同一首歌，不同音源的歌词来源不同。

**Q10：无感过渡（Crossfade）开启后切歌时有明显断点？**

A：无感过渡需要提前预加载下一首歌曲的音频流，网络较慢时可能预加载不足导致衔接不流畅。可适当降低音质或在网络良好的环境下使用。

---

## 四、播放列表

**Q11：双击歌曲后不播放，只是把歌曲加入了列表末尾，这是 BUG 吗？**

A：这是预设的设计行为：双击将歌曲添加到播放列表末尾，不会立即切换播放。若需立即播放，请点击歌曲左侧的 ▶ 图标，将其插入列表开头并立即播放。

**Q12：播放列表如何导出或备份？**

A：进入「设置 → 播放 → 播放列表管理」，点击「导出」可将当前播放列表导出为文件或复制到剪贴板。导出的内容可在其他设备上通过「导入」功能恢复。注意：本地音源歌曲会被自动筛除，仅导出在线音源歌曲。

**Q13：播放列表中的歌曲顺序如何调整？**

A：展开播放列表面板，对目标歌曲长按约 1.5 秒后进入拖拽模式，将其拖动到目标位置后松开。列表顺序实时保存到本地。

## 五、下载问题

**Q14：某些歌曲的下载按钮是灰色的，无法点击？**

A：该歌曲受版权限制，当前音源不提供可下载的链接。可尝试切换到其他音源搜索同一首歌，看是否有下载权限。

**Q15：下载完成后文件在哪里？**

A：点击「下载管理器 → 已完成」标签页中该任务右侧的 📁 文件夹图标，可直接定位到文件。也可在「设置 → 存储 → 下载目录」中查看当前保存路径。

**Q16：下载速度很慢，如何提升？**

A：下载速度主要受音源服务器限速和本地网络影响，可以：① 尝试切换到其他音源；② 检查网络带宽；③ 降低所下载的音质，文件大小更小相对下载更快。

## 六、本地音乐

**Q17：添加了本地目录但列表是空的？**

A：等待扫描进度完成（进度条在页面顶部显示）；如果已完成，点击刷新图标重新扫描。确认所添加目录下存有 mp3、flac、wav 等受支持的音频格式文件。

**Q18：本地音乐没有封面，如何添加？**

A：右键点击歌曲选择「匹配音频」，应用将在线匹配封面和歌词。若匹配结果不准确，可通过「编辑标签」手动上传封面图片。

---

## 七、账号与个人信息

**Q19：如何修改账号密码或邮箱？**

A：进入「个人信息（Profile）」页面，在「账号安全」区域点击对应的修改选项。操作将跳转到系统默认浏览器中的澜音认证服务完成，整个过程密码与凭据不经过本应用。

**Q20：修改头像后没有更新？**

A：确认图片上传完成（保存按钮点击后等待加载完成），部分情况需要重启应用或刷新页面才能看到新头像。支持 jpg、png 等常见图片格式。

---

## 八、其他问题

**Q21：应用更新如何操作？**

A：Ceru Music 支持自动检测更新。检测到新版本后，会在下载管理器中出现「应用更新」类型的任务。下载完成后点击「立即安装」，应用将自动退出并安装新版本。

**Q22：应用最小化到托盘后找不到图标？**

A：Windows 系统可能将托盘图标隐藏到「显示隐藏的图标」列表中（任务栏右侧的 ^ 按钮）。macOS 系统在菜单栏右侧可以看到澜音图标。

**Q23：如何彻底退出应用（而不是最小化）？**

A：如果「关闭按钮行为」设置为「最小化到托盘」，点击窗口关闭按钮只会最小化到托盘。可以在托盘图标右键菜单中选择「退出」彻底关闭应用，或在「设置 → 外观」中将关闭行为改为「直接退出」。

**Q24：在哪里可以反馈 BUG 或提建议？**

A：请前往以下渠道提交反馈：
- GitHub Issues：[https://github.com/timeshiftsauce/CeruMusic](https://github.com/timeshiftsauce/CeruMusic)
- 官方 Q 群：[加入 Q 群](/guide/source/qq_group)
