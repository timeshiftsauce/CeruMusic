# 歌曲下载使用指南

## 一、文档概述

- **文档目的：** 指导用户在 Ceru Music 中对歌曲执行下载操作，并通过下载管理器跟踪任务进度。
- **功能简介：** 支持对搜索结果或歌单中的歌曲发起下载，下载管理器提供进度查看、暂停/继续/取消/重试，以及已完成文件的定位打开。
- **适用平台：** Windows、macOS 桌面客户端
- **适用用户：** 所有普通用户

## 二、使用前提

- 已安装 Ceru Music（澜音）最新版本。
- 已启动应用并完成插件（音源）导入，且当前音源支持所选歌曲的下载。
- 在「设置 → 存储 → 下载目录」中已确认或设置下载保存路径。

## 三、功能说明

| 功能 | 说明 |
|------|------|
| 发起下载 | 对单首歌曲触发下载请求 |
| 下载队列 | 支持同时上限 1～5 个任务并行下载 |
| 暂停/继续 | 可随时暂停正在进行的下载并在需要时继续 |
| 取消下载 | 取消进行中或排队中的任务 |
| 重试下载 | 对失败的任务重新发起下载 |
| 打开文件位置 | 下载完成后快速定位到文件所在目录 |
| 文件名模板 | 可自定义下载文件的命名格式 |
| 并发数设置 | 调整同时进行的最大下载任务数（1～5） |

## 四、操作步骤

### 4.1 下载单首歌曲

**方式一：右键菜单下载**

1. 在搜索结果页、歌单详情页或播放列表中，右键单击目标歌曲。
2. 在弹出的上下文菜单中选择「下载」。
3. 选择下载音质（如有多种音质可选）。
4. 任务将自动添加到下载队列，下载管理器中可查看进度。

**方式二：歌曲详情下载**

1. 在歌曲列表中找到目标歌曲，点击展开歌曲详情或相关操作区域。
2. 点击下载图标发起下载。

### 4.2 查看下载管理器

1. 点击左侧导航栏中的「下载」，进入下载管理器页面。
2. 页面顶部显示三个标签页：
   - **进行中：** 显示正在下载、排队等待及已暂停的任务。
   - **已完成：** 显示下载成功的任务，按创建时间降序排列。
   - **失败/已取消：** 显示发生错误或已手动取消的任务。

### 4.3 管理下载任务

| 操作 | 方法 |
|------|------|
| 暂停单个任务 | 在「进行中」标签页，点击该任务右侧的 ⏸ 暂停图标 |
| 继续暂停的任务 | 点击该任务右侧的 ▶ 继续图标 |
| 取消任务 | 点击该任务右侧的 ✕ 取消图标（仅限进行中和排队中） |
| 重试失败任务 | 在「失败/已取消」标签页，点击该任务右侧的 ↺ 重试图标 |
| 打开文件所在目录 | 在「已完成」标签页，点击该任务右侧的 📁 文件夹图标 |
| 删除任务记录 | 在已完成或失败标签页，点击 🗑 删除图标移除记录（不删除本地文件） |
| 全部开始 | 在「进行中」标签页顶部，点击「全部开始」批量恢复所有暂停任务 |
| 全部暂停 | 点击「全部暂停」批量暂停所有进行中的任务 |
| 清空队列/记录 | 点击「清空队列」或「清空记录」清除当前标签页全部内容 |

### 4.4 设置并发下载数

1. 在下载管理器页面右上角，找到「同时下载数」设置。
2. 使用数字输入框将并发数设置为 1～5（默认为 3）。
3. 修改后立即生效，后续任务按新的限制调度。

### 4.5 自定义文件名模板

1. 进入「设置 → 存储」页面。
2. 找到「下载文件名格式」设置项，输入自定义模板。
3. 模板变量说明：

| 变量 | 含义 |
|------|------|
| `%t` | 歌曲名称 |
| `%s` | 歌手名称 |
| `%a` | 专辑名称 |

默认模板为 `%t - %s`，即「歌曲名 - 歌手名」。

## 五、操作结果

- 下载任务创建成功：下载管理器「进行中」标签页出现新任务条目，显示歌曲名称、状态及进度条。
- 下载完成：任务自动移入「已完成」标签页，进度显示 100%，状态标签变为绿色「完成」。
- 文件定位：点击 📁 后系统资源管理器（或 Finder）自动打开并选中对应文件。

## 六、注意事项

- 部分歌曲因版权限制不支持下载，右键菜单中不会显示下载选项，或下载时报错。
- 下载文件保存到本地后，卸载应用**不会**自动删除已下载的文件。
- 更换设备后下载的文件不会自动迁移，需在新设备上重新下载。
- 取消或删除任务记录不会删除已保存到本地的文件，如需删除请手动操作文件系统。
- 下载队列按添加时间先进先出（FIFO）调度，可通过暂停其他任务变相提高特定任务的优先级。

## 七、常见问题

**Q1：右键菜单中没有下载选项，是什么原因？**

A：该歌曲受版权限制不支持下载，或当前音源插件未提供该歌曲的可下载链接。可尝试切换音源后重新搜索该歌曲。

**Q2：下载任务一直处于「排队等待」状态，不开始下载？**

A：当前正在下载的任务数已达到并发上限，等待前面的任务完成后会自动开始。如需加快进度，可适当提高「同时下载数」的设置值。

**Q3：下载失败，错误信息是「获取链接失败」，如何处理？**

A：通常是音源插件未能返回有效的下载地址。建议：① 检查网络连接；② 切换到其他音源重新搜索并下载；③ 稍后重试。

**Q4：下载完成后找不到文件在哪里？**

A：点击「已完成」标签页中该任务右侧的 📁 文件夹图标，可直接定位到文件所在位置。也可以在「设置 → 存储 → 下载目录」中查看当前下载目录路径，手动前往该目录查找。

**Q5：下载的文件能否在其他播放器中播放？**

A：下载的文件为标准音频格式（如 mp3、flac 等），可用其他支持该格式的播放器正常播放。

**Q6：如何修改下载文件的保存位置？**

A：进入「设置 → 存储」，在「下载目录」设置项中更改保存路径。修改后仅对新增的下载任务生效，已完成的文件不会移动。
