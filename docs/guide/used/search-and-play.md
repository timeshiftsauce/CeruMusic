# 搜索与播放使用指南

## 一、文档概述

- **文档目的：** 指导用户在 Ceru Music 中完成搜索歌曲、浏览发现页以及控制音乐播放的全部操作。
- **功能简介：** 搜索功能支持按关键词搜索歌曲与歌单；发现页提供排行榜与分类歌单浏览；播放控制支持暂停/继续、上下曲切换、进度调节、音量调节及多种播放模式；无感过渡播放（Crossfade）在切换歌曲时自动平滑衔接音频。
- **适用平台：** Windows、macOS 桌面客户端
- **适用用户：** 所有普通用户

## 二、使用前提

- 已安装 Ceru Music（澜音）最新版本。
- 已启动应用并完成插件（音源）导入。若尚未导入音源，请先参考 [安装教程 - 插件安装](/guide/) 完成音源配置，否则无法获取在线音乐数据。

## 三、功能说明

| 功能 | 说明 |
|------|------|
| 搜索歌曲 | 按关键词搜索，结果分页加载，支持切换音源 |
| 搜索歌单 | 在搜索结果页可切换「歌单」标签查看歌单结果 |
| 搜索联想 | 在搜索框输入时自动展示候选词，提升输入效率 |
| 发现页 | 展示排行榜及按分类浏览的推荐歌单 |
| 播放控制 | 播放/暂停、上一首/下一首、进度拖拽、音量调节 |
| 播放模式 | 顺序播放、随机播放、单曲循环 |
| 全屏播放 | 进入沉浸式全屏播放界面，显示歌词、专辑封面、频谱可视化 |
| 无感过渡播放 | 当前曲目结束前自动与下一首进行淡入淡出衔接 |

## 四、操作步骤

### 4.1 搜索歌曲

1. 在顶部导航栏的搜索框中输入歌曲名、歌手名或关键词。
2. 输入过程中会自动展示搜索联想词，可直接点击候选词快速搜索。
3. 按下 `Enter` 键或点击搜索图标提交搜索。
4. 搜索结果页默认展示「歌曲」标签，列表中显示歌曲名、歌手、专辑及时长。
5. 向下滚动列表，系统将自动加载更多结果（每页 50 条）。

> **切换音源：** 点击搜索框左侧的音源图标，可在已安装的插件音源之间快速切换，结果将根据所选音源重新加载。

### 4.2 搜索歌单

1. 完成搜索后，在搜索结果页顶部点击「歌单」标签。
2. 搜索结果将切换为歌单列表，显示歌单封面、名称及歌曲数量。
3. 点击歌单封面或名称，进入歌单详情页。

### 4.3 浏览发现页

1. 点击左侧导航栏中的「发现」进入发现页。
2. 页面顶部展示当前音源的**排行榜**，点击对应排行榜卡片可查看榜单歌曲。
3. 页面下方展示按**分类**划分的推荐歌单：
   - 点击分类标签（如「热门」「流行」「说唱」等）切换分类。
   - 向下滚动自动加载更多歌单。
4. 点击任意歌单封面进入歌单详情页。

### 4.4 播放歌曲

| 操作 | 方式 |
|------|------|
| 添加到播放列表末尾并播放 | 双击歌曲名称 |
| 插入到播放列表开头并立即播放 | 点击歌曲左侧的 ▶ 图标 |
| 右键歌曲 | 展开上下文菜单，可选择播放方式、下载、分享等操作 |

### 4.5 播放控制

播放控制栏位于窗口**底部**，包含以下控件：

| 控件 | 操作 |
|------|------|
| 播放/暂停 | 点击中央的 ▶/⏸ 按钮（应用内快捷键：`Space`） |
| 上一首 | 点击 ⏮ 按钮 |
| 下一首 | 点击 ⏭ 按钮 |
| 进度调节 | 拖拽进度条或点击进度条任意位置（应用内快捷键：`←` 快退 5 秒，`→` 快进 5 秒） |
| 音量调节 | 拖拽音量滑块（应用内快捷键：`↑` +5%，`↓` -5%） |
| 播放模式 | 点击播放模式图标循环切换（顺序 / 随机 / 单曲循环） |
| 全屏播放 | 点击播放封面或按应用内快捷键 `F` |

### 4.6 全屏播放

1. 点击播放控制栏左侧的歌曲封面，或按应用内快捷键 `F`，进入全屏播放界面。
2. 全屏播放界面包含：
   - 专辑封面展示（可在设置中切换黑胶唱片/圆形封面样式）
   - 实时滚动歌词
   - 实时频谱可视化（可在设置中开关）
   - 背景律动动画（可在设置中开关）
3. 再次按 `F` 或点击右上角关闭按钮退出全屏。

### 4.7 无感过渡播放（Crossfade）

无感过渡播放会在当前歌曲即将结束时自动与下一首进行淡入淡出混音，实现平滑切换，无需手动操作。

启用后，切换歌曲时屏幕边缘会短暂显示「正在平滑过渡」提示。

**开启方式：** 进入「设置 → 播放 → 播放列表管理」，开启「无感过渡播放」开关。

## 五、操作结果

- 搜索完成：搜索结果列表正常加载，显示对应歌曲或歌单。
- 播放成功：底部播放控制栏显示当前播放歌曲信息，进度条开始运动。
- 进入全屏：界面切换到全屏播放视图，歌词同步滚动。
- 无感过渡激活：切换歌曲时出现「正在平滑过渡」提示，音频无明显断点。

## 六、注意事项

- 搜索与发现功能依赖已安装的音源插件，未安装插件时将无法获取在线音乐。
- 切换音源后搜索结果将重新加载，不同音源的曲库内容可能有所不同。
- 全屏播放中的背景动画与频谱可视化会占用较高 CPU/GPU 资源，低配设备可在设置中关闭。
- 无感过渡播放开启后，手动切换上/下一首同样会触发过渡效果。

## 七、常见问题

**Q1：搜索时提示无法获取数据或列表为空，怎么处理？**

A：检查以下几点：① 确认已在设置中导入并启用音源插件；② 切换到其他音源后重试；③ 检查网络连接是否正常。

**Q2：为什么双击歌曲后没有立即播放？**

A：默认双击行为是将歌曲添加到播放列表末尾，不会立即切换到该歌曲播放。若需立即播放，请点击歌曲左侧的 ▶ 图标，将其插入列表开头并立即播放。

**Q3：搜索联想词没有出现怎么办？**

A：搜索联想依赖音源插件提供的接口，如果当前音源不支持联想词功能或网络延迟较高，联想词可能不显示，可直接按 `Enter` 提交搜索。

**Q4：发现页的分类歌单加载失败怎么处理？**

A：当前音源可能暂时不支持分类歌单接口，可尝试切换到其他音源后重新进入发现页。

**Q5：全屏播放时歌词不同步怎么办？**

A：部分歌曲的歌词时间轴可能存在偏差，这属于音源插件提供的歌词数据问题，无法在应用内修正。可在全屏播放界面检查歌词是否正常显示，如无歌词则说明该歌曲暂无可用歌词数据。

**Q6：无感过渡播放开启后感觉有杂音或卡顿，如何处理？**

A：无感过渡播放在切换时需同时解码两路音频流，对设备性能有一定要求。建议在设置中适当降低音频品质，或关闭背景动画、频谱可视化等高耗能视觉效果。
