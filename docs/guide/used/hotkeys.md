# 快捷键使用指南

## 一、文档概述

- **文档目的：** 列出 Ceru Music 的所有可用快捷键，并指导用户自定义全局快捷键。
- **功能简介：** 提供两类快捷键：全局快捷键（在任何窗口下均可触发）和局内快捷键（仅在应用窗口聚焦时生效）。全局快捷键支持自定义绑定。
- **适用平台：** Windows、macOS 桌面客户端
- **适用用户：** 所有普通用户

## 二、使用前提

- 已安装 Ceru Music（澜音）最新版本。
- 配置全局快捷键需在「设置 → 快捷键」中启用「全局快捷键」开关。

## 三、快捷键列表

### 3.1 全局快捷键（可自定义）

全局快捷键在系统范围内生效，即使 Ceru Music 窗口不在前台或最小化，也可通过快捷键控制播放。

| 功能 | 默认快捷键 | 说明 |
|------|-----------|------|
| 播放/暂停 | 可自定义 | 切换当前播放状态 |
| 上一首 | 可自定义 | 切换到播放列表中的上一首歌曲 |
| 下一首 | 可自定义 | 切换到播放列表中的下一首歌曲 |
| 快退 5 秒 | 可自定义 | 当前歌曲进度后退 5 秒 |
| 快进 5 秒 | 可自定义 | 当前歌曲进度前进 5 秒 |
| 桌面歌词显示/隐藏 | 可自定义 | 切换桌面歌词的可见状态 |
| 音量 -5% | 可自定义 | 全局将应用音量降低 5% |
| 音量 +5% | 可自定义 | 全局将应用音量提升 5% |
| 切换为顺序播放 | 可自定义 | 将播放模式设为顺序播放 |
| 切换为随机播放 | 可自定义 | 将播放模式设为随机播放 |
| 单曲循环切换 | 可自定义 | 在单曲循环与顺序播放之间切换 |

> **说明：** 应用安装后全局快捷键默认均为空（未绑定），需要用户手动设置。

### 3.2 局内快捷键（固定不可更改）

局内快捷键仅在 Ceru Music 主窗口聚焦且光标不在文本输入框内时生效。

| 快捷键 | 功能 |
|--------|------|
| `Space`（空格键） | 播放/暂停，切换当前播放状态 |
| `↑`（方向键上） | 音量 +5% |
| `↓`（方向键下） | 音量 -5% |
| `←`（方向键左） | 快退 5 秒 |
| `→`（方向键右） | 快进 5 秒 |
| `F` | 切换全屏播放面板（进入/退出全屏播放） |

## 四、设置全局快捷键

### 4.1 进入快捷键设置

1. 点击右上角设置图标进入「设置」页面。
2. 在左侧分类中点击「快捷键」，进入快捷键设置界面。

### 4.2 启用 / 禁用全局快捷键

1. 在页面顶部找到「启用全局快捷键」开关。
2. 开启后，已绑定的全局快捷键将注册到系统级，在任何窗口下均可触发。
3. 关闭后，所有全局快捷键将注销，不再响应。

### 4.3 为功能绑定快捷键

1. 在快捷键列表中找到目标功能行（如「播放/暂停」）。
2. 点击该行右侧的「设置」按钮，弹出录入对话框。
3. 在对话框中按下想要绑定的组合键（如 `Ctrl + Alt + P`），对话框中会实时显示捕获到的按键。
4. 确认无误后点击「保存」，快捷键立即生效。

> **录入提示：**
> - 按 `Esc` 取消录入并关闭对话框。
> - 按 `Backspace` 或 `Delete` 清空当前输入。
> - 若所按组合键已被其他功能占用，会提示「快捷键已被其它功能占用」，需换用其他组合键。

### 4.4 清除快捷键绑定

1. 在目标功能行右侧点击「清除」按钮。
2. 该功能的快捷键绑定立即清除，不再触发该功能。

### 4.5 恢复默认快捷键

1. 在快捷键列表上方点击「恢复默认」按钮。
2. 所有功能的快捷键将恢复到默认绑定配置，同时立即保存并生效。

## 五、操作结果

- 绑定成功：对应功能行右侧的标签更新为新快捷键的显示名称（如 `Ctrl+Alt+P`），颜色正常为蓝色。
- 绑定失败：对应功能行标签显示为红色，悬浮提示注册失败原因（通常为与系统其他应用冲突）。
- 清除成功：功能行右侧标签变为空或显示「未设置」。

## 六、注意事项

- 全局快捷键在系统级注册，若其他应用已占用相同组合键，注册可能失败，建议使用较少见的三键组合（如 `Ctrl + Alt + 字母`）。
- 局内快捷键（如 `Space`、方向键）在文本输入框内不触发，避免与正常输入冲突。
- 快捷键配置持久保存，重启应用后自动恢复。
- macOS 系统对某些快捷键有限制（如 `Command + Q`），建议避免使用。

## 七、常见问题

**Q1：设置了全局快捷键但没有反应，是什么原因？**

A：检查以下几点：① 确认「设置 → 快捷键」中「启用全局快捷键」开关已开启；② 确认该快捷键行右侧标签为正常蓝色（而非红色报错）；③ 检查是否有其他进程占用了相同快捷键；④ 尝试重启应用重新注册。

**Q2：快捷键标签显示红色，提示注册失败，如何解决？**

A：该快捷键与系统或其他应用发生冲突，无法注册。请点击「设置」按钮重新输入不同的组合键，或关闭占用该快捷键的其他应用后重试。

**Q3：按空格键没有触发播放/暂停，怎么处理？**

A：局内快捷键要求 Ceru Music 主窗口处于聚焦（最前台）状态，且当前焦点不在任何文本输入框中。请点击应用窗口的非输入区域后再尝试按空格键。

**Q4：可以绑定媒体键（如键盘上的播放按钮）吗？**

A：支持绑定大多数媒体键，在录入对话框中按下媒体键即可识别。部分键盘的媒体键可能被系统或驱动拦截，导致无法识别，需尝试关闭系统媒体键全局监听。

**Q5：为什么不能设置单个字母键（如只按 `P`）作为全局快捷键？**

A：单个字母键会与正常文字输入冲突，应用要求全局快捷键必须包含功能修饰键（如 `Ctrl`、`Alt`、`Shift`、`Command`）组合，不允许注册纯字母键。
