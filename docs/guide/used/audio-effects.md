# 音效与均衡器设置指南

## 一、文档概述

- **文档目的：** 指导用户在 Ceru Music 中调整音频均衡器（EQ）和高级音效处理，优化听感体验。
- **功能简介：** 提供图形均衡器调节各频段增益，以及高级音效模块（低音增强、环绕音效、声道平衡）的开关与参数控制。
- **适用平台：** Windows、macOS 桌面客户端
- **适用用户：** 希望优化音质体验的普通用户

## 二、使用前提

- 已安装 Ceru Music（澜音）最新版本。
- 设备音频输出正常工作。
- 进入「设置 → 播放」找到「音频均衡器」和「高级音效处理」设置组。

## 三、功能说明

| 功能模块 | 说明 |
|----------|------|
| 音频均衡器（EQ） | 对多个频段独立调节增益，改变音色特性 |
| 低音增强（Bass Boost） | 提升低频音量，增强打击感 |
| 环绕音效（Surround） | 模拟不同空间的混响效果 |
| 声道平衡（Balance） | 调整左右声道音量比例 |

## 四、操作步骤

### 4.1 音频均衡器（EQ）

均衡器通过对不同频段进行增减来改变音色。各频段从左到右频率依次升高，覆盖低频（低音）到高频（高音/气声）。

1. 进入「设置 → 播放 → 音频均衡器」。
2. 拖拽各频段的垂直滑块，向上拖动为正增益（增强该频段），向下为负增益（削弱该频段）。
3. 修改后立即生效，播放中的音乐会实时变化。
4. 点击「重置」可将所有频段归零。

**常见均衡器预设参考：**

| 风格 | 建议调整方式 |
|------|------------|
| 流行（清晰人声） | 低中频略提升，高频稍提升，中频段平坦 |
| 摇滚/电子（冲击感） | 低频明显提升，中高频稍降，超高频略提 |
| 古典/爵士（自然） | 全频段基本平坦，或轻微提高高频细节 |
| 纯净听音 | 所有频段归零（平直响应） |

### 4.2 低音增强（Bass Boost）

低音增强专门针对低频段进行提升，适合听电子音乐、嘻哈、EDM 等风格时使用。

1. 进入「设置 → 播放 → 高级音效处理」，找到「低音增强 (Bass Boost)」卡片。
2. 打开右上角的开关以启用。
3. 使用「低频增益」滑块调整增益值（范围 -12 dB 至 +12 dB）：
   - 正值为增强，负值为削弱。
   - 增益为 0 时效果与关闭等同。
4. 或使用预设按钮快速套用：

| 预设 | 增益值 |
|------|--------|
| 轻度 | +3 dB |
| 中度 | +6 dB |
| 重度 | +9 dB |

### 4.3 环绕音效（Surround）

环绕音效通过模拟不同空间的混响，在普通立体声耳机或音响上产生虚拟空间感。

1. 找到「环绕音效 (Surround)」卡片，打开开关。
2. 选择环境模式：

| 模式 | 效果描述 |
|------|---------|
| 关闭 | 不应用任何空间效果 |
| 小房间 | 小空间感，混响短促，贴近真实室内 |
| 中厅堂 | 中等空间感，适合流行、摇滚 |
| 大教堂 | 宽阔空间混响，适合古典、纯音乐 |

3. 切换模式后立即生效，无需保存。

> **提示：** 环绕音效模拟 5.1/7.1 虚拟声场与环境混响，效果最佳时配合头戴式耳机使用。

### 4.4 声道平衡（Balance）

声道平衡用于补偿左右耳听力差异，或满足特定的声场偏好。

1. 找到「声道平衡 (Balance)」卡片，打开开关。
2. 拖拽平衡条滑块：
   - 向左拖：左声道音量增大，右声道减小；
   - 向右拖：右声道音量增大，左声道减小；
   - 居中时两声道均等。
3. 点击「居中校准」按钮可一键恢复到 Center（0）。
4. 数值说明：
   - 显示为 `Left XX%` 表示偏向左侧，百分比为偏移量
   - 显示为 `Right XX%` 表示偏向右侧
   - 显示为 `Center` 表示左右均等

### 4.5 重置所有音效

点击「高级音效处理」卡片右上角的「重置全部」按钮，可将低音增强、环绕音效及声道平衡所有参数恢复默认值（全部关闭，增益归零，平衡居中）。

## 五、操作结果

- 均衡器调整：播放中的音乐音色立即变化，低频/高频段的响度随滑块位置实时改变。
- 低音增强：低频打击感增强（或减弱），在节奏强的音乐中效果明显。
- 环绕音效：声场扩展，产生空间混响感，在封闭空间内使用耳机时效果最为明显。
- 声道平衡：左右耳听到的音量比例变化。

## 六、注意事项

- 均衡器各频段增益范围通常为 ±12 dB，大幅提升可能导致音频削波（爆音），建议增益值不超过 +9 dB。
- 低音增强与均衡器可同时使用，但叠加效果下低频过度增益容易失真。
- 环绕音效对立体声音源（普通耳机、立体声音箱）最为有效，单声道音源效果有限。
- 声道平衡只影响应用内的音频输出，不修改系统级的声道设置。
- 更换音频设备（如从耳机切换到音箱）后，建议重新调整均衡器和音效参数，因为不同设备对频率响应差异较大。

## 七、常见问题

**Q1：开启低音增强后声音出现破音/失真，怎么办？**

A：低音增益值过高导致音频信号超出范围发生削波。建议将低音增益降低到 +6 dB 以下，并同时检查系统音量是否过高。

**Q2：均衡器调整后感觉没有变化？**

A：确认均衡器已开启（非全部归零状态），且当前播放有音频输出。部分音乐本身频率特性明显，变化相对不易察觉，可将低频段设置到极值（+12 dB 或 -12 dB）验证均衡器是否生效。

**Q3：环绕音效开启后音量变小了？**

A：环绕效果处理会对整体音量有轻微衰减，属于正常现象，适当提高系统音量或应用音量补偿即可。

**Q4：声道平衡保存后重启应用会不会失效？**

A：设置会自动保存，重启应用后上次的参数状态将自动恢复。

**Q5：找不到均衡器设置项在哪里？**

A：进入「设置」后，点击左侧「播放」分类，向下滚动可以看到「音频均衡器」和「高级音效处理」两个设置组。
