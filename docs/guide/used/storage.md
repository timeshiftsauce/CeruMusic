# 数据存储设置指南

## 一、文档概述

- **文档目的：** 指导用户在 Ceru Music 中配置缓存目录与下载目录的存储路径，管理本地缓存，以及自定义下载时的文件命名规则和标签写入选项。
- **功能简介：** 存储设置涵盖缓存目录与下载目录的路径选择、目录大小查看、缓存清理、自动缓存策略、下载文件名模板，以及下载时嵌入音频文件的标签内容控制。
- **适用平台：** Windows、macOS 桌面客户端
- **适用用户：** 所有普通用户

## 二、使用前提

- 已安装 Ceru Music（澜音）最新版本。
- 进入方式：点击右上角设置图标 → 左侧分类选择「存储」。

## 三、功能说明

| 功能模块 | 说明 |
|----------|------|
| 存储目录配置 | 分别设置缓存文件和下载文件的保存路径 |
| 本地歌曲缓存 | 查看缓存占用大小并一键清除 |
| 缓存策略 | 控制是否在播放时自动读写缓存 |
| 下载文件名格式 | 自定义下载歌曲的文件名模板 |
| 下载标签写入 | 控制下载时向音频文件写入的元数据内容 |

## 四、默认存储路径

应用首次运行后会自动在系统标准位置创建以下目录，无需手动操作：

| 目录 | Windows 默认路径 | macOS 默认路径 |
| ---- | ---------------- | -------------- |
| **缓存目录** | `C:\Users\<用户名>\AppData\Roaming\澜音\music-cache` | `~/Library/Application Support/澜音/music-cache` |
| **下载目录** | `C:\Users\<用户名>\Music\CeruMusic\songs` | `~/Music/CeruMusic/songs` |
| **应用配置文件** | `C:\Users\<用户名>\AppData\Roaming\澜音\sqj_config.json` | `~/Library/Application Support/澜音/sqj_config.json` |

> **说明：**

- `AppData\Roaming` 是 Windows 系统的应用数据隐藏目录，可在文件资源管理器地址栏输入 `%AppData%` 直接跳转。
- macOS 的 `~/Library` 目录默认隐藏，可在 Finder 中按 `Shift + Command + G` 输入路径跳转，或在终端中访问。
- 以上路径均可在「设置 → 存储 → 存储目录配置」中自定义修改。

## 五、操作步骤

### 5.1 配置存储目录

存储目录配置分为**缓存目录**和**下载目录**两部分，均在「设置 → 存储 → 存储目录配置」卡片中管理。

#### 缓存目录

缓存目录用于存放应用播放时自动缓存的音频文件，加速重复播放时的加载速度。

1. 当前缓存目录路径显示在路径输入框中（只读）。
2. 旁边的标签显示当前缓存目录已占用的磁盘空间。
3. **更改缓存目录：**
   - 点击「选择目录」按钮，在系统弹出的文件夹选择对话框中选择目标文件夹。
   - 选择完成后路径输入框更新为新路径，提示「缓存目录已选择，记得保存」。
   - 点击页面底部的「保存设置」按钮使修改生效。
4. **打开缓存目录：** 点击「打开目录」按钮，系统文件管理器直接定位到该目录。

#### 下载目录

下载目录用于存放通过应用下载的音乐文件。

1. 当前下载目录路径显示在路径输入框中（只读）。
2. 旁边的标签显示当前下载目录已占用的磁盘空间。
3. **更改下载目录：**
   - 点击「选择目录」按钮，选择目标文件夹。
   - 选择完成后提示「下载目录已选择，记得保存」。
   - 点击「保存设置」按钮使修改生效。
4. **打开下载目录：** 点击「打开目录」按钮快速定位到目录。

#### 重置为默认目录

点击「存储目录配置」卡片右上角的「重置为默认」按钮，将两个目录路径恢复到系统默认位置。确认弹窗后立即生效、自动保存，无需再点「保存设置」。

> **注意：** 目录更改后记得点击「保存设置」，否则关闭设置页面修改将丢失。

### 4.2 管理本地歌曲缓存

「本地歌曲缓存配置」卡片显示当前已缓存文件的总大小及文件数量。

**清除缓存：**

1. 点击「清除本地缓存」按钮。
2. 弹出确认对话框提示「这可能会导致歌曲加载缓慢，确定要清除吗？」。
3. 点击「确定清除」后，应用开始执行清除操作，进度按钮显示「正在清除...」。
4. 清除完成后，缓存大小显示归零，后续播放需重新缓存音频文件。

> **注意：** 清除缓存不会删除已下载的歌曲文件，仅清除播放时产生的临时缓存。

### 4.3 设置缓存策略

在「缓存策略」区域找到「自动缓存音乐」开关：

| 状态 | 行为 |
|------|------|
| 开启 | 播放在线音乐时自动写入本地缓存，下次播放同一首歌时优先读取缓存，提升加载速度 |
| 关闭 | 不进行自动缓存，每次播放都从网络加载，节省磁盘空间但增加流量消耗 |

切换开关后立即保存，无需额外操作。

### 4.4 设置下载文件名格式

「下载文件名格式设置」区域允许自定义下载歌曲时保存的文件名规则。

**可用模板变量：**

| 变量 | 含义 | 示例 |
|------|------|------|
| `%t` | 歌曲名称 | 半岛铁盒 |
| `%s` | 歌手名称 | 周杰伦 |
| `%a` | 专辑名称 | 八度空间 |
| `%u` | 平台来源 | tx |
| `%q` | 音质标识 | 320k / flac / master |
| `%d` | 下载日期 | 2026-01-01 |

**操作步骤：**

1. 在模板输入框中输入自定义格式，可自由组合变量和分隔符。
2. 输入框下方会实时显示预览效果（以示例歌曲「半岛铁盒 - 周杰伦」为例）。
3. 修改后自动保存，新格式对后续下载任务生效，已完成的任务文件不会重命名。

**常用格式示例：**

| 模板 | 示例结果 |
|------|---------|
| `%t - %s`（默认） | 半岛铁盒 - 周杰伦 |
| `%s - %t` | 周杰伦 - 半岛铁盒 |
| `%t - %s [%q]` | 半岛铁盒 - 周杰伦 [flac] |
| `%s/%a/%t` | 周杰伦/八度空间/半岛铁盒（可用斜杠创建子目录层级） |

### 4.5 下载标签写入设置

「下载标签写入设置」控制下载歌曲时向音频文件的元数据（标签）中写入哪些信息。

| 选项 | 说明 | 默认状态 |
|------|------|---------|
| 基础信息 | 写入歌曲标题、艺术家、专辑名称等基本信息 | 开启 |
| 封面 | 将专辑封面图片嵌入到音频文件中 | 开启 |
| 歌词信息 | 将歌词写入音频文件的元信息字段 | 开启 |
| 单独下载歌词文件 | 在下载目录中额外保存一个独立的 `.lrc` 歌词文件 | 关闭 |

**歌词格式选项**（在「歌词信息」或「单独下载歌词文件」至少一项开启时可用）：

| 格式 | 说明 |
|------|------|
| 标准 LRC 歌词 | 行级时间轴，兼容性最强，适合大多数播放器 |
| 逐字歌词 | 精确到每个字的时间轴，支持逐字高亮，需要播放器支持此格式 |

卡片底部「当前配置」一栏实时显示已开启的标签选项汇总，便于确认。

## 五、操作结果

- 目录保存成功：提示「目录设置已保存」，后续操作（下载/缓存）使用新路径。
- 缓存清除成功：卡片标题处的缓存大小更新为 0 B，文件数归零。
- 文件名模板修改：预览行实时更新，下载管理器中新建任务的文件名按新模板显示。
- 标签写入成功：下载完成后用标签编辑工具（如 Windows 文件属性或第三方工具）可查看嵌入的信息。

## 六、注意事项

- 更改存储目录后，**已缓存**或**已下载**的文件**不会**自动迁移到新目录，需要手动移动。
- 清除缓存后再次播放同一首歌曲时加载速度可能变慢，建议在磁盘空间不足时再操作。
- 文件名模板中若包含斜杠（`/`），下载时会在下载目录内按层级自动创建子目录。
- 歌词格式选择「逐字歌词」时，若音源插件不提供逐字歌词数据，会自动回退到标准 LRC 格式。
- 单独下载的 `.lrc` 歌词文件与音频文件同名，保存在同一下载目录中，如需手动关联请确保文件名一致。

## 七、常见问题

**Q1：选择了新的下载目录后，已下载的歌曲还能找到吗？**

A：已下载的文件仍然保存在原来的目录中，不会自动移动。如需整合，请手动将原目录中的文件复制或移动到新目录。

**Q2：缓存目录和下载目录可以设置为同一个文件夹吗？**

A：可以，但不推荐。两者混用会使文件管理更混乱，缓存文件（临时性）和下载文件（持久保存）最好分开存放。

**Q3：清除缓存后，下载的歌曲还在吗？**

A：下载的歌曲文件不受影响，清除缓存只会删除播放时自动产生的临时缓存文件（保存在缓存目录中），下载目录的文件完整保留。

**Q4：下载的歌曲在 Windows 文件属性中看不到封面或歌词，是什么原因？**

A：确认「下载标签写入设置」中「封面」和「歌词信息」已勾选。若之前下载时未开启，已有文件不会补写标签，需重新下载。也可使用第三方标签编辑工具（如 Mp3tag）查看完整标签信息。

**Q5：「单独下载歌词文件」生成的 `.lrc` 文件在哪里查看？**

A：`.lrc` 文件与音频文件保存在同一下载目录中，文件名与音频文件相同（仅扩展名不同），可用文本编辑器打开查看，支持 LRC 格式的播放器也可自动加载。

**Q6：保存目录设置时提示「保存失败」，如何处理？**

A：检查所选目录是否存在且应用有写入权限。macOS 用户需确认已授予澜音访问对应目录的权限（系统偏好设置 → 安全性与隐私 → 文件和文件夹）。
