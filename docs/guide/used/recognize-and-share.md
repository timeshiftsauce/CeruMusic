# 听歌识曲与歌曲分享使用指南

## 一、文档概述

- **文档目的：** 指导用户使用 Ceru Music 的听歌识曲功能识别未知歌曲，以及将歌曲或歌单分享给他人。
- **功能简介：** 听歌识曲支持通过麦克风录音或上传音频片段识别歌曲；歌曲分享功能可生成带有效期的分享链接，歌单分享支持分享整个播放列表。
- **适用平台：** Windows、macOS 桌面客户端
- **适用用户：** 所有普通用户

## 二、使用前提

**听歌识曲：**
- 已安装 Ceru Music（澜音）最新版本。
- 设备麦克风可用（录音识别方式需要）。
- 已登录账号（需要后端服务处理识别请求）。
- 网络连接正常。

**歌曲分享：**
- 已安装 Ceru Music（澜音）最新版本。
- 需要联网（分享链接由后端服务生成）。
- 已理解并同意分享功能说明（首次使用需在分享对话框中确认授权）。

## 三、功能说明

### 听歌识曲

| 功能 | 说明 |
|------|------|
| 麦克风录音识别 | 通过麦克风录取环境中正在播放的音频片段（最长 15 秒），上传识别 |
| 结果展示 | 显示识别到的候选歌曲列表，含封面、歌曲名、歌手信息 |
| 直接播放 | 从识别结果列表直接播放对应歌曲 |
| 搜索跳转 | 将识别结果的歌曲名作为关键词跳转到搜索页 |

### 歌曲与歌单分享

| 功能 | 说明 |
|------|------|
| 分享单曲 | 生成包含歌曲信息的分享链接，有效期 1～7 天可选 |
| 分享歌单 | 分享整个歌单，接收方可通过链接导入歌单 |
| 有效期设置 | 分享链接在设定时间后自动失效，保护隐私 |
| 授权确认 | 首次分享时需确认已阅读分享功能使用条款 |

## 四、操作步骤

### 4.1 使用麦克风识别歌曲

1. 在左侧导航栏中找到「识曲」入口（或在菜单中找到「听歌识曲」），进入识曲页面。
2. 确保录音设备（麦克风、手机扬声器等）旁边有正在播放的歌曲。
3. 点击页面中央的「开始识别」（麦克风图标）按钮，应用开始通过麦克风录音。
4. 录音最长 15 秒，采集足够音频片段后（通常 5～10 秒即可），应用自动停止录音并开始上传处理。
5. 等待识别结果（通常几秒内返回）：
   - 识别成功：显示候选歌曲列表，含封面、歌曲名、歌手。
   - 识别失败：提示无法识别，可重试或换用其他方式。
6. 点击结果列表中的某首歌曲右侧的 ▶ 图标可直接播放。
7. 点击搜索图标可将歌曲名跳转到搜索页查找更多信息。

### 4.2 分享单首歌曲

1. 在搜索结果、歌单详情或播放列表中，右键单击目标歌曲。
2. 在上下文菜单中选择「分享」。
3. 弹出「分享歌曲」对话框：
   - 顶部显示歌曲封面、歌曲名、歌手、专辑等信息。
   - 「有效期」滑块：拖动设置链接有效期，范围 1～7 天。
4. 首次使用时，勾选页面底部的授权确认条款。
5. 点击「分享」按钮，系统生成分享链接。
6. 链接生成后，点击「复制链接」将链接复制到剪贴板，发送给对方。

### 4.3 分享歌单

1. 在左侧「歌单」列表中右键单击目标歌单，或进入歌单详情页。
2. 点击「分享歌单」按钮，弹出分享歌单对话框。
3. 设置有效期并确认授权后点击「分享」。
4. 分享链接生成后复制并发送给对方。接收方使用 Ceru Music 打开链接后，可导入该歌单。

## 五、操作结果

**听歌识曲：**
- 识别成功：页面展示候选歌曲列表，包含封面、名称、歌手信息。
- 识别失败：提示「识别失败」，可重新尝试或换用其他音频片段。

**歌曲分享：**
- 链接生成成功：对话框状态从「设置」切换为进度步骤展示，全部步骤完成后显示「分享成功」。
- 链接复制成功：粘贴到聊天工具后可正常打开。

## 六、注意事项

**听歌识曲：**
- 在安静环境中录音效果更好；背景噪音过大会降低识别准确率。
- 录音过程中应用会暂停当前播放（若正在播放），识别完成后恢复，避免麦克风拾取到自身播放干扰。
- 识别服务依赖网络请求，弱网环境下可能超时失败。

**歌曲分享：**
- 分享链接含有效期，超过设定时间后链接自动失效，对方无法再访问。
- 分享的歌曲信息（名称、歌手、封面）会上传到澜音服务器用于展示，请勿分享含敏感信息的内容。
- 接收方需安装 Ceru Music 才能通过链接播放歌曲。
- 本地歌曲（source 为 local）不支持分享。

## 七、常见问题

**Q1：识曲时麦克风无法使用，提示权限不足，如何处理？**

A：

- **Windows：** 打开「系统设置 → 隐私 → 麦克风」，确认已允许桌面应用访问麦克风，并确认 Ceru Music 在应用列表中已开启。
- **macOS：** 打开「系统偏好设置 → 安全性与隐私 → 麦克风」，在应用列表中勾选澜音。

**Q2：已经录了很久仍无法识别，怎么办？**

A：确认录音的音频片段包含清晰的旋律（非间奏、纯静音或噪音段），必要时换用包含明确旋律的歌曲段落重试。另外检查网络连接是否正常。

**Q3：识别结果与实际歌曲不符，如何处理？**

A：识别结果依赖第三方指纹识别精度，对于翻唱、改编、纯音乐等内容识别准确率可能较低。可以通过歌曲的关键词手动搜索来查找正确版本。

**Q4：分享对话框一直转圈，链接没有生成，是什么原因？**

A：分享功能依赖澜音后端服务，若服务暂时不可用或网络不稳定可能导致请求超时。等待片刻后刷新重试，或检查网络连接。

**Q5：发送给朋友的分享链接对方打不开，怎么办？**

A：① 确认链接有效期未过；② 确认对方已安装 Ceru Music（澜音）；③ 确认发送的链接完整未被截断（部分聊天工具会截断长链接，建议使用纯文本粘贴）。

**Q6：收到别人分享的歌单链接，如何导入？**

A：在 Ceru Music 中打开链接（或点击链接后自动调用打开），应用会弹出「导入歌单」对话框，确认后歌单内歌曲将合并到当前播放列表或创建新歌单。
